¿Tienes alguna pregunta?

  • ¿Cómo creo una playlist?

    Las playlists son ideales para guardar colecciones de música, ya sea para disfrutarlas solo o para compartirlas con tus amigos. Para crear una:

    1. Ve a tu biblioteca
    2. Selecciona Crear
    3. Ponle un nombre a tu playlist
    4. Empieza a agregar canciones (nosotros te ayudaremos)
  • ¿Cómo activo el modo Ahorro de datos?
    1. Ve a Inicio
    2. Dirigete a la Configuración
    3. Elige Ahorro de datos
    4. Activa el ahorro de datos
  • ¿Solo se puede escuchar música en modo aleatorio?

    Las playlists que tengan el ícono de reproducción aleatoria se reproducirán en modo aleatorio.
    Algunas playlists no tendrán el ícono de reproducción aleatoria, así que podrás elegir la canción que quieras para escuchar.

  • ¿Dónde encuentro podcasts?

    Selecciona Buscar. En Explorar todo,elige Podcast.
